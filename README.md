# Booking-App
This web app lets users to book the company's facilities such as conference rooms.

App Features:

1. View all current bookings in a timeline chart (Day vs Time).
2. See detailed information on all existing reservations in a table format.
3. Submit a new booking request for a conference room.
4. Receive an automated confirmation email after the booking.
5. The system prevents overlapping bookings for the same room.
